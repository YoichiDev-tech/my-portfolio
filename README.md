# Yoichi – Frontend Portfolio

A focused, frontend‑heavy portfolio for my indie founder career.  
Built to showcase prototypes, MVPs, and launched products with a clear, honest collaboration model.


## Tech stack

- React
- TypeScript
- Vite
- CSS (custom, neon‑dark + glassmorphism aesthetic)


## Features

- **Landing page** with clear positioning and CTAs
- **Collaboration / Work With Me** page describing how you work
- **Getting Started** process with structured steps
- **Portfolio** with:
  - Prototypes & MVPs (hidden until user clicks “View prototype work”)
  - Launched projects (hidden until user clicks “View launched work”)
- **Contact** and **Book a Call** flows (integrated with Cal.com)
- Agreement card explaining project start conditions (signed agreement + 35% deposit)


## Portfolio behavior

- **Prototypes & MVPs**
  - Section is visible
  - Items are hidden until the user clicks **“View prototype work”**
  - Card: **WOF Form — Prototype** with live link + GitHub link

- **Launched projects**
  - Section is visible
  - Items are hidden until the user clicks **“View launched work”**
  - Card: **Cafe — Launched Project** with live link + GitHub link

## Agreement & project start

Before any work begins:

- A short agreement is reviewed and signed.
- Scope, responsibilities, timelines, and a **35% upfront deposit** are defined.
- Once signed and paid, the project officially starts.

This keeps expectations clear and protects both sides.
